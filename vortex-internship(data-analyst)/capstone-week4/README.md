# VortexTech Data Science & Analytics Internship - Week 4 (Capstone Project)

## 📊 End-to-End Customer Churn Analysis Report

A comprehensive data science project investigating customer churn patterns, analyzing key drivers, and providing actionable business recommendations.

### Key Highlights
- **Executive Summary:** Focuses on mitigating customer churn through proactive onboarding and support optimization.
- **Visualizations:** 5+ professional charts illustrating churn distribution, subscription types, support calls impact, and customer tenure.
